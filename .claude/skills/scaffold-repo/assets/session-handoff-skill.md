# session-handoff

## Workflow

Copy this checklist and track progress:

```
- [ ] Step 1: Run scripts/status.sh — get session snapshot
- [ ] Step 2: Update docs/next.md
- [ ] Step 3: Update docs/roadmap.md (if phase status changed)
- [ ] Step 4: Close finished GitHub issues
- [ ] Step 5: Write memory for anything non-obvious
- [ ] Step 6: Ask about archiving the session
```

See references/checklist.md for the detailed procedure for each step.

## Boundaries

- Do not skip steps — all six run every session
- docs/next.md is for the build only — personal items go elsewhere
- Memory is for non-obvious decisions only, not a recap of what happened
