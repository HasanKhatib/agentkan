# Release v1 — MVP

## Goal

A shippable first version users can sign up for and use daily.

## Exit criteria

- [ ] A new user signs up with email and password
- [ ] Onboarding wizard completes and lands on a populated home screen
- [ ] App is reachable over HTTPS on the production domain
- [ ] Core action works end to end without support

## Out of scope

- Email digests and retention features (v2)
- Advanced analytics beyond sign-up counts
