### Design principles documented

- Hybrid JSON state + markdown bodies (not pure JSON, not pure markdown)
- AI proposes, human disposes (agent never marks epics done without ask)
- Single-file viewer, zero dependencies, File System Access API on localhost
- Git is history; stable epic ids for small diffs; viewer sorts by id
